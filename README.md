# dp · doble página

Primer sitio web personal diseñado en diciembre de 2003. 

El diseño de este sitio avanza sobre la idea de las *[transpointing windows](http://xanadu.com.au/ted/TN/PARALUNE/paraviz.html)* definicas por [Ted Nelson](https://es.wikipedia.org/wiki/Ted_Nelson) en un formato de doble página, caracterizando los lados izquierdo y derecho como facetas independientes en una lectura intencionada de hipertexto.

Este sitio presenta la materia tratada en los talleres de la Escuela de Arquitectura y Diseño PUCV por **Herbert Spencer**. 

Para mayor información remítase al colofón.
